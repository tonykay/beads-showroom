== Learning Outcomes

By completing this course you will be able to:

. Install and initialize Beads in any project
. Use `bd create`, `bd ready`, `bd update`, and `bd close` as your daily task loop
. Configure Claude Code to prime Beads context automatically at session start
. Keep your Beads installation healthy with `bd doctor`, cleanup, and upgrades
. *(Bonus)* Connect Beads to a GitHub repo and link issues to pull requests

== Prerequisites

* Comfortable with git and a terminal
* No Beads or Dolt experience needed
* Claude Code recommended (required for Module 3) — free tier works
* macOS, Linux, or Windows WSL

[source,sh]
----
